## Git Workflow

- 작업마다 별도의 feature branch를 생성한다.
- 파일 수정이 포함된 작업은 반드시 PR 생성까지 완료한다.
- 작업 완료 후 PR을 생성하는 것까지만 수행한다.
- PR의 merge 및 close 여부는 사용자가 직접 결정한다. Claude가 merge하거나 close하지 않는다.
