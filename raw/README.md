# raw/

Unprocessed knowledge dumps — notes, ideas, research, anything worth capturing.

Files here are source material for the wiki. Run `/wiki-ingest` in Claude Code
to incorporate them into `src/content/wiki/`.

## Naming

Use descriptive lowercase filenames: `llm-context-windows.md`, `swiftui-state-notes.md`.
No frontmatter needed — plain prose, bullets, or code snippets.

## Lifecycle

1. Drop a `.md` file here
2. Add it to `wiki-log.md` under `## Unincorporated` with today's date
3. Run `/wiki-ingest` to process it
